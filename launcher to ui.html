📋 Update Log
⚠ Warnings
  • Worst Time Sim removed temporarily due to space constraints on the new UI.
  • Only gray and black backgrounds currently work on the new UI.
  • Native keyboard locking in games is not available in the new UI.
📰 News
  • Old UI now has keyboard and cursor locking in games.
  • Old UI has been reverted back 2 versions.
  • New UI is, well, new.
  • Opening games in a new tab now works in the new UI.
  • Submit requests: Open Form ↗
⌨ CMD Fixes
  • Fixed exit — was calling a broken selector, now uses closeW(win).
  • Full rewrite — replaced innerHTML += with proper appendChild.
  • Fixed title, cd, and color commands.
  • Single-instance: reopening CMD focuses the existing window.
🎮 Games
  • Cookie Clicker: replaced dead URL with the real site.
  • Poly Track: loads via blob URL (sss.html) — bypasses iframe restrictions.
  • Poly Track resizes correctly when the window is moved or maximized.
  • Maximize now maximizes within the UI instead of opening a blank tab.
  • Pop-out button (↗) added to every game titlebar.
  • Blocked iframes show a clean "Open in new tab" fallback.
🖥 Fullscreen & Taskbar
  • Taskbar hides automatically when any window is maximized.
  • Maximized windows fill full 100vh — no gap at the bottom.
  • Taskbar restores when the window is unmaximized or closed.
  • Dock mode: taskbar shrinks to content, centered with rounded corners.
  • App icons and tray each get their own frosted pill in dock mode.
  • Height presets: Slim / Normal / Large / XL — all icons scale proportionally.
  • All taskbar settings persist via localStorage.
🗑 Removed
  • Auto-hide On/Off toggle removed — taskbar hides on fullscreen automatically.
  • All fake Windows processes removed (System, Explorer.exe, Dwm.exe, etc.).
  • Ghost Settings/CMD entries visible when those apps were closed are gone.
📊 Task Manager
  • UI row is permanent — shows real JS heap memory, no End button.
  • Every open app shows tracked memory with an End button.
  • Games labeled "game"; system apps labeled "system".
  • Process count is accurate — no fake +42 anymore.
  • Memory values are stable — random jitter removed.
  • Ending a process now correctly restores the taskbar if needed.
